# JupiteerOS
Experimental Operating System

## Build requirements
In order to build JupiteerOS, make sure to have the following installed:
- git
- bash
- make
- GCC 
- nasm
- QEMU (to test it)

### Build
```
make
```
### Compile to ISO
```
make iso
```
### Run in QEMU:
```
make run
```
### Run in Bochs
```
make bochs
```
After running bochs, type "c" to start it.
